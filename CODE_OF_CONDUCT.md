# Code of Conduct

## Nuestro compromiso

Este proyecto busca mejorar la comprension publica de datos electorales. La participacion debe ser respetuosa, verificable y no partidaria.

## Conducta esperada

- Citar fuentes primarias cuando afirmes hechos electorales.
- Separar resultados oficiales de analisis, escenarios y opiniones.
- Tratar desacuerdos metodologicos con evidencia y buena fe.
- Cuidar la privacidad de votantes, personeros, funcionarios y colaboradores.

## Conducta no aceptada

- Acoso, insultos, ataques personales o propaganda coordinada.
- Desinformacion electoral deliberada.
- Publicacion de datos personales, credenciales o material privado.
- Presentar simulaciones como resultados oficiales.

## Aplicacion

El mantenedor puede cerrar issues, editar o eliminar comentarios y bloquear participantes cuando una interaccion ponga en riesgo la integridad civica, la seguridad o la privacidad.
